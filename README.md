# .NET Excel Sales Report Generator using ClosedXML

![.NET Version](https://img.shields.io/badge/.NET-10-blueviolet)
![License](https://img.shields.io/badge/License-MIT-green)

A sample .NET 10 project demonstrating how to generate a complex, multi-sheet Excel sales report using the powerful **ClosedXML** library. This project is designed to showcase a common backend task: automating business intelligence reports directly from application data, with **zero dependency on Microsoft Office**.

### ClosedXML Features Demonstrated

* Creating multiple worksheets (tabs).
* Applying rich cell styling (background colors, bold fonts, alignment).
* Merging cells (`Merge()`) for titles.
* Setting data number formats (Currency, Dates, Numbers).
* Auto-adjusting column widths (`AdjustToContents()`).
* Creating formatted tables.

Follow these instructions to clone and run the project on your local machine.

### Prerequisites

* [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)

### ğŸš€ Running the Project

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ElCabronw/ExcelWithClosedXml.git
    cd ExcelWithClosedXml
    ```

2.  **Restore .NET dependencies:**
    ```bash
    dotnet restore
    ```

3.  **Run the application:**
    ```bash
    dotnet run
    ```

4.  **Check the output:**
    After running, a new file (e.g., `Sales_Report_YYYYMMDD_HHMMSS.xlsx`) will be created in the project's output folder (e.g., `bin/Debug/net10.0`).

    Open this file with Excel, Google Sheets, or any spreadsheet reader to see the result.
